# 一、平台部署说明 #
------------------------
> 1. 数据库配置文件：src/conf/db.properties
> 2. 安装mysql数据库，创建用户和数据库。
> 3. 运行mysql.sql文件。
> 4. 如需要oauth2的，设置src/conf/oauth.properties
